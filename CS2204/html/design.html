Theme: Mainly use cold tonal colors. The background is a view of the sea, which correspond to the users' feeling of relax on their vacation.
The font type is also informal (cursive) to relax users.
The layout of the webpage is mainly developed by frameset tags. The values are set in a relative way (use percentage).
Banner: Use the modular style. The components of the banners are buttons that link to these several pages.
Using the hover design to make some change on these buttons "on_mouse_over" and "gradient"
Page 1 (search.html): The video will play automatically. "Find a hotel", "Things to do" are two large blocks, in which smaller blocks are used ("Skiing","Diving") The calendar is developed by table. The information input is within the frame
Page 2 (hotel.html): The "Search" part is roughly the same as it is in the first page, except that it is used more like a navigation.
The right hind side is 3 independent blocks about 3 hotels. Source of the pictures are here
Page 3 (room.html): Here we automatically assume that the first (Recommended) hotel is selected.
Those input columns are in the same "form" tag.
A horizontal line "hr" is used to devided the page into two parts.
The lower part is also made up with 4 blocks.
The "Book" button are in the forms. Type: submit
Page 4 (confirm.html): As the booking information cannot be handled without the use of Java Script, I randomly writhe some information there.
The confirm button is also in a form, action="search.html"(default page)
Below which a link "here" is pointed back to the room page for making new changes.
